# Tech Stack — real-estate-maps

## Linguagem
- Python 3.11+

## Scraping
- **Playwright** — automação de browser para Google Maps
- `playwright install chromium`

## Banco de Dados
- **Supabase** (PostgreSQL) — armazenamento das agências
- `supabase-py` — cliente Python

## Dashboard
- **Streamlit** — interface web
- **Pandas** — manipulação de dados
- **Pydeck** — mapa interativo

## CLI
- `argparse` — interface de linha de comando

## Tabela Principal
- `real_estate_agencies` no Supabase
